About

The Decentralised Discovery Gateway is a service offering plug-and-play message and data federation across different nodes. It features a fully decentralised architecture with minimal trust, where each node controls user authorisation autonomously. It also allows nodes to exist beyond a project or cohort's lifetime, even after the original operators have gone or the central portal has been shut down.

The networks created by Decentralised Discovery Gateway nodes are local -- means that they are not controlled by any central authority. As a result, it's often hard to find -- you need to know which node to check and what network you want to join. This registry aims to solve that problem by allowing services running a compatible protocol implementation to register their networks and nodes here, making them discoverable and accessible to users from outside their local projects.
